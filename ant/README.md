ant
===

Apache Ant is a Java library and command-line tool whose mission is to drive
processes described in build files as targets and extension points dependent
upon each other.

Samples
-------
```
include ant
```
```
class { 'ant':
  version => '1.9.6',
  source  => 'http://www.us.apache.org/dist/ant/binaries'
}
```
