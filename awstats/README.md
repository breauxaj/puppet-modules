awstats
=======

AWStats is a free powerful and featureful tool that generates advanced web,
streaming, ftp or mail server statistics, graphically.

Samples
-------
```
include awstats
```
```
class { 'awstats':
  version => '7.4',
  source  => 'http://prdownloads.sourceforge.net/awstats'
}
```
