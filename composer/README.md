composer
========

Composer is a tool for dependency management in PHP. It allows you to declare
the dependent libraries your project needs and it will install them in your
project for you.

Samples
-------
```
include composer
```
```
class { 'composer':
  source => 'https://getcomposer.org/composer.phar'
}
```
