modsecurity
===========

ModSecurity is a toolkit for real-time web application monitoring, logging, and
access control.

Samples
-------
```
include modsecurity
```
```
include modsecurity::crs
```
```
class { 'modsecurity':
  version => 'latest'
}
```
```
class { 'modsecurity::crs':
  version => 'latest'
}
```
