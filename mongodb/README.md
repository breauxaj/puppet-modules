mongodb
=======

MongoDB (from "humongous") is an open-source document database, and the
leading NoSQL database.

Samples
-------
```
include mongodb
```
```
mongodb::config {
  'bind_ip': value => '127.0.0.1';
  'port':    value => '27017';
  'dbpath':  value => '/var/lib/mongodb';
}
```
