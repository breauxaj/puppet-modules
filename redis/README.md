redis
=====

Redis is an open source, BSD licensed, advanced key-value store. It is often
referred to as a data structure server since keys can contain strings, hashes,
lists, sets and sorted sets.

Samples
-------
```
include redis
```
```
redis::config {
  'port':   value => '6379';
  'bind':   value => '192.168.0.5';
}
```
